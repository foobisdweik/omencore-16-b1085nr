Assets for RGB device images and brand assets

Guidelines:
- Use PNG or SVG files in the brand folders (Corsair, Razer, Logitech)
- Prefer 2x resolution images (e.g., 512x512) and include an SVG when possible
- Include an ATTRIBUTION.md if using vendor-provided assets that require permission
- Filenames should be lowercase, hyphenated (e.g., k70-rgb-mk2.png)
